# Wiki or Whack? 🎉
**Wikipedia 25th Anniversary Trivia Game**

Real-time multiplayer — up to 40+ players, one big screen host.

---

## Files
- `index.html` — Players open this on their phones
- `host.html` — You open this on the big screen
- `README.md` — This file

---

## Setup on GitHub Pages (free hosting)

1. Go to **github.com** → sign in → click **New repository**
2. Name it `wiki-or-whack` → set to **Public** → **Create repository**
3. Click **Add file → Upload files**
4. Upload both `index.html` and `host.html`
5. Click **Commit changes**
6. Go to **Settings → Pages → Source → Deploy from branch → main → Save**
7. Wait ~1 minute. Your URLs will be:
   - Players: `https://YOURUSERNAME.github.io/wiki-or-whack/`
   - Host: `https://YOURUSERNAME.github.io/wiki-or-whack/host.html`

---

## On event day

1. Open `host.html` on your laptop connected to the big screen (fullscreen with F11)
2. The player URL is shown on screen — put it in a QR code at **qr.io** (free)
3. Print or display the QR code so players can scan and join
4. Players type their name and tap **Join Game**
5. Watch the player count go up on the host screen
6. When everyone is in — hit **Start Game**
7. You control the pace — click **Next Question** after each reveal
8. Final leaderboard shows automatically at the end 🏆

---

## Game rules
- Each question: players tap **That's a FACT** or **Total WHACK**
- Score = up to 10 points, based on how fast they answer
- Answer in 1s = 10 pts · Answer in 15s = ~2 pts · No answer = 0 pts
- 3 out of every 5 questions are real Wikipedia facts (some flipped to false)
- 2 out of 5 are AI-generated statements
- After each question: Wikipedia summary + link shown to everyone

---

## To reset between rounds
Click **Reset Game** on the host screen. All scores and players clear.
